# SiglusSceneScriptUtility GUI 规格说明

**版本：** 0.1（草案）  
**语言：** 仅中文界面  
**上游 CLI：** [siglus-ssu](https://github.com/Jirehlov/SiglusSceneScriptUtility)  
**参考手册：** [manual_cn.md](manual_cn.md)

---

## 1. 目标

为 `siglus-ssu` 命令行工具提供一个**简单易上手**的图形界面，让不熟悉终端的用户也能完成常见操作：

- 通过**文件/文件夹选择器**指定输入与输出路径，无需手打路径
- 按功能分区，每个命令对应一个清晰的面板
- 常用选项以复选框、下拉框呈现；高级选项默认折叠
- 实时显示命令输出日志，失败时给出可读的中文提示

GUI **不重新实现**编译/提取逻辑，统一通过子进程调用 `siglus-ssu`，保证与 CLI 行为一致。

---

## 2. 技术约束

| 项目 | 说明 |
|---|---|
| Python | ≥ 3.12（与上游一致） |
| 依赖 | 复用现有 `siglus-ssu` 包；GUI 使用标准库 `tkinter`（无额外依赖） |
| 调用方式 | `subprocess` 执行 `siglus-ssu`（开发环境可用 `uv run siglus-ssu`） |
| 平台 | Windows 优先；路径分隔符使用 `pathlib` 处理 |

### 3.1 通用控件规范

| 控件 | 行为 |
|---|---|
| **输入路径** | 根据功能显示「选择文件」或「选择文件夹」按钮；支持拖放（可选，二期） |
| **输出路径** | 文件类输出用「另存为」；目录类输出用「选择文件夹」；可留空以使用 CLI 默认行为 |
| **浏览按钮** | 标签统一为「浏览…」 |
| **路径框** | 只读或可编辑（建议可编辑，方便粘贴路径） |
| **开始执行** | 校验必填项后禁用按钮、启动子进程、流式追加日志 |
| **打开输出目录** | 执行成功后可用；用系统文件管理器打开输出所在目录 |
| **运行日志** | 等宽字体；支持复制、清空；运行中自动滚到底部 |

### 3.2 全局选项（窗口底部或设置菜单）

| GUI 标签 | CLI 参数 | 说明 |
|---|---|---|
| 纯 Python 模式（禁用 Rust 加速） | `--legacy` | 调试用 |
| 常量配置 | `--const-profile N` | 下拉：0 / 1 / 2，默认 0 |
| 关于 / 版本 | `-V` | 只读显示 |

## 4. 功能面板规格

侧边栏顺序按**使用频率**排列：提取 → 编译 → … → 工具类。

---

### 4.1 提取（`-x` / `--extract`）

**用途：** 从 `.pck` 提取场景；反汇编/反编译 `.dat`；还原 `Gameexe.ini`。

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 操作类型 | 单选 | 是 | ① 提取 .pck　② 反汇编 .dat（目录或 .pck）　③ 还原 Gameexe.ini |
| 输入 | 文件/文件夹 | 是 | ① `.pck`　② `.pck` 或含 `.dat` 的文件夹　③ `Gameexe.dat` 或其所在目录 |
| 输出目录 | 文件夹 | 否 | 留空则输出到输入同目录（CLI 默认） |
| 同时反汇编并反编译 | 复选框 | 否 | `--disam`（操作类型 ① 时可用） |
| 密钥来源 | 文本框 + 浏览 | 否 | `--angou`；可填游戏目录、`angou=文本`、`key=十六进制` |

**CLI 映射示例：**

```
siglus-ssu -x [--disam] <input> [output_dir] [--angou VALUE]
siglus-ssu -x --gei <Gameexe.dat|dir> [output_dir] [--angou VALUE]
```

---

### 4.2 编译（`-c` / `--compile`）

**用途：** 将 `.ss` 源码目录编译为 `.pck`；或单独编译 `Gameexe.dat`。

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 操作类型 | 单选 | 是 | ① 标准编译　② 仅编译 Gameexe.dat |
| 输入目录 | 文件夹 | 是 | 含 `.ss` 或 `Gameexe.ini` 的工作目录 |
| 输出 | 文件或文件夹 | 是 | `.pck` 文件路径，或已存在目录（在其中生成 `Scene.pck`） |
| 源文件编码 | 下拉 | 否 | 自动 / UTF-8 / Shift-JIS（`--charset`） |
| 保留临时文件 | 复选框 | 否 | `--debug` |
| 串行编译 | 复选框 | 否 | `--serial` |
| 增量编译缓存目录 | 文件夹 | 否 | 填写则启用 `--tmp`（高级，可折叠） |
| 仅重打包已有 .dat | 复选框 | 否 | `--dat-repack` |
| 禁用加密 | 复选框 | 否 | `--no-angou` |
| 禁用 LZSS | 复选框 | 否 | `--no-lzss` |

**CLI 映射示例：**

```
siglus-ssu -c [选项] <input_dir> <output_pck|output_dir>
siglus-ssu -c --gei <input_dir|Gameexe.ini> <output_dir>
```

---

### 4.3 分析（`-a` / `--analyze`）

**用途：** 查看 `.pck` / `.dat` 等内部结构；比较两个文件；统计台词字数。

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 操作类型 | 单选 | 是 | ① 分析单个文件　② 比较两个文件　③ 台词统计导出 CSV　④ 分析 Gameexe.dat |
| 输入文件 1 | 文件 | 是 | 支持 `.pck` `.dat` `.gan` `.sav` `.cgm` `.tcr` 等 |
| 输入文件 2 | 文件 | 否 | 比较模式必填 |
| 输出 CSV | 文件 | 否 | 台词统计模式；留空则 CLI 默认路径 |
| 写出反汇编 | 复选框 | 否 | `--disam` |
| 语义级比较 | 复选框 | 否 | `--payload`（比较模式，较慢） |
| 密钥来源 | 文本框 | 否 | `--angou` |

**说明：** 分析结果主要输出到日志区（stdout），不单独生成 GUI 报告页。

### 4.4 图片 g00（`-g` / `--g00`）

**用途：** 提取、分析、合并、创建 `.g00` 图片资源。

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 操作 | 下拉 | 是 | 分析 `--a` / 提取 `--x` / 合并 `--m` / 创建 `--c` |
| 输入 | 文件或文件夹 | 是 | 依操作类型：`.g00`、图片目录、PNG/JPEG 等 |
| 输入 2+ | 文件列表 | 否 | 合并模式：多个 `.g00`（界面提供「添加文件」） |
| 输出目录/文件 | 文件或文件夹 | 视操作 | 提取、创建、合并时必填或建议填写 |
| 裁剪透明边 | 复选框 | 否 | `--trim`（提取/合并） |
| g00 类型 | 数字 | 否 | 创建时 `--type N` |
| 参考 g00 | 文件 | 否 | 创建时 `--refer` |

### 4.9 文本映射（`-m` / `--textmap`）

**用途：** 翻译工作流——从 `.ss` 或 `.dat` 导出 CSV，译后写回。

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 操作 | 单选 | 是 | ① 从 .ss 导出　② 写回 .ss　③ 从 .dat 导出　④ 写回 .dat |
| 输入 | 文件或文件夹 | 是 | `.ss` / `.dat` 或其目录 |
| 输出 | 文件夹 | 否 | 导出时默认与输入同目录生成 `.csv` |

| GUI 操作 | CLI |
|---|---|
| 从 .ss 导出 | `siglus-ssu -m <path>` |
| 写回 .ss | `siglus-ssu -m --apply <path>` |
| 从 .dat 导出 | `siglus-ssu -m --disam <path>` |
| 写回 .dat | `siglus-ssu -m --disam-apply <path>` |

## 5. 命令组装规则

1. 全局选项放在子命令之前：`siglus-ssu [--legacy] [--const-profile N] <mode> ...`
2. 路径含空格时用引号包裹（`subprocess` 列表形式传参，无需 shell 引号）
3. 输出路径留空时**不传**该参数，交给 CLI 默认行为
4. 互斥选项在 GUI 层校验（如 `--tmp` 与 `--debug` 不可同时勾选），与 [manual_cn.md](manual_cn.md) 一致
5. 工作目录：默认 GUI 程序所在目录或用户上次使用的目录；不强制 `chdir` 到输入目录

## 6. 错误与状态

| 情况 | GUI 行为 |
|---|---|
| 必填项为空 | 「开始执行」前阻止，中文提示缺哪一项 |
| 路径不存在 | 执行前校验，提示「找不到文件/文件夹」 |
| 子进程非零退出 | 状态栏显示失败；日志保留；不禁用「开始执行」 |
| 长时间任务 | 日志流式更新；可选显示不确定进度条 |
| const.py 缺失 | 启动时检测，引导运行「初始化」 |

## 9. 典型用户流程（验收用例）

### 9.1 汉化提取 → 编译

1. **提取**：选 `Scene.pck`，输出到 `D:\work\game_cn\`，勾选「反汇编并反编译」
2. 在外部编辑器修改 `decompiled/*.ss` 或导出的文本
3. **文本映射**（可选）：导出 CSV → 翻译 → 写回
4. **编译**：输入 `D:\work\game_cn\`，输出 `D:\work\SceneZH.pck`

### 9.2 仅提取语音

1. **语音收集**：场景选 `Scene.pck`，语音目录选游戏 `voice`，输出选空文件夹

### 9.3 引擎中文化

1. **引擎补丁**：`--lang cjk-path`，选 `SiglusEngine.exe`，指定输出或勾选覆盖
