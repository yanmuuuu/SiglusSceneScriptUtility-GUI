# SiglusSceneScriptUtility GUI 规格说明

**版本：** 0.2  
**语言：** 仅中文界面  
**上游 CLI：** [siglus-ssu](https://github.com/Jirehlov/SiglusSceneScriptUtility)  
**用户手册：** [instructions.md](instructions.md)  
**CLI 手册：** [manual_cn.md](manual_cn.md)

---

## 1. 目标

为 `siglus-ssu` 提供**清晰、响应快**的图形界面：

- 分组侧边导航 + 功能说明，降低 15 个命令的学习成本
- **路径 / 选项** 分区展示；按操作类型**动态隐藏**无关字段
- 高级参数默认折叠，避免界面杂乱
- 日志**批量刷新**（约 80ms 一批），避免 CLI 大量输出时界面卡顿
- 面板**懒加载**，加快冷启动
- 统一通过子进程调用 `siglus-ssu`，行为与命令行一致

---

## 2. 技术约束

| 项目 | 说明 |
|---|---|
| Python | ≥ 3.12 |
| 依赖 | 标准库 `tkinter` / `ttk`；无第三方 GUI 库 |
| 调用 | `subprocess` + 后台线程读 stdout；列表形式传参 |
| 平台 | Windows 优先（Segoe UI、无控制台窗口）；其他平台回退系统字体 |
| 性能 | 日志队列 + 定时 flush；日志上限约 40 万字符；LSP stdout 后台 drain |

### 3.1 导航分组

| 分组 | 功能 |
|---|---|
| **场景与脚本** | 提取、编译、分析、文本映射、场景教程、执行标签 |
| **游戏资源** | 图片 g00、音频、视频、数据库、语音收集 |
| **系统与工具** | 引擎补丁、初始化、回编测试、语言服务器 |

当前选中项高亮（`NavSelected.TButton`）。

### 3.2 通用控件

| 控件 | 行为 |
|---|---|
| **浏览…** | 文件 / 文件夹 / 另存为；记住上次目录 |
| **路径框** | 可编辑，支持粘贴 |
| **字段 hint** | 标签下方灰色小字说明格式与留空规则 |
| **开始执行** | 校验 → 禁用按钮 → 不确定进度条 → 流式日志 |
| **停止** | `taskkill /T` 终止子进程树 |
| **打开输出目录** | 用资源管理器打开最近一次成功任务的输出位置 |
| **运行日志** | 深色主题；右键复制 / 全选；`Ctrl+L` 清空 |
| **快捷键** | `Ctrl+Enter` 执行当前功能 |

### 3.3 全局选项（窗口底部）

| GUI 标签 | CLI 参数 | 说明 |
|---|---|---|
| 纯 Python 模式 | `--legacy` | 禁用 Rust 扩展，排查用 |
| 常量配置 | `--const-profile N` | 0 / 1 / 2，默认 0 |

## 4. 功能面板规格

侧边栏顺序与 `PANEL_ORDER` 一致。各面板顶部说明见 `PANEL_HINTS`。

### 4.1 提取（`-x`）

| 字段 | 说明 |
|---|---|
| 操作类型 | ① 提取 .pck　② 反汇编 .dat　③ 还原 Gameexe.ini |
| 输入 / 输出目录 | 输出可留空 |
| 同时反汇编并反编译 | `--disam`（仅操作 ① 时显示） |
| 密钥来源 | `--angou`（可选） |

### 4.2 编译（`-c`）

| 区域 | 字段 |
|---|---|
| 路径 | 输入目录、输出（.pck 或目录） |
| 常用选项 | 源文件编码、--debug、--serial |
| 高级（折叠） | --dat-repack、--no-angou、--no-lzss、--tmp |

互斥：`--debug` 与 `--tmp` 不可同时启用。

### 4.3 分析（`-a`）

操作类型切换时动态显示字段：

| 操作 | 显示字段 |
|---|---|
| 分析单个文件 | 输入 1、--disam、密钥 |
| 比较两个文件 | 输入 1+2、--disam、--payload |
| 台词统计 CSV | 输入 1、输出 CSV |
| 分析 Gameexe.dat | 输入 1、密钥 |

### 4.4–4.15 其余面板

与 v0.1  CLI 映射相同；**图片 g00 / 音频 / 视频** 按操作类型隐藏无关路径与选项。详见 [instructions.md](instructions.md) 分步说明。

## 5. 命令组装规则

1. `siglus-ssu [--legacy] [--const-profile N] <子命令> …`
2. `subprocess` 列表传参，路径含空格无需手动引号
3. 输出留空则不传对应参数
4. GUI 层校验互斥选项与必填项
5. 执行命令行在日志首行以 `$ …` 显示

---

## 6. 错误与状态

| 情况 | GUI 行为 |
|---|---|
| 必填项为空 / 路径不存在 | 弹窗中文提示，不启动子进程 |
| 子进程非零退出 | 状态栏显示退出码；日志保留 |
| 长时间任务 | 不确定进度条；日志批量刷新 |
| const.py 缺失 | 启动后引导至「初始化」 |
| 关闭窗口时任务运行中 | 确认后终止子进程 |
