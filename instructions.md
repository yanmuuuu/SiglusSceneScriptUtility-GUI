## 便携版（推荐）

**无需安装 Python、uv 或任何运行环境。**

1. 下载 `SiglusSSU-GUI-portable.zip` 并解压
2. 将 `SiglusSSU-GUI` 文件夹放到桌面
3. 双击 **`SiglusSSU-GUI.exe`**

```
桌面/
└── SiglusSSU-GUI/
    ├── SiglusSSU-GUI.exe    ← 双击这个
    ├── 启动 SiglusSSU-GUI.bat
    ├── 使用说明.txt
    └── _internal/           ← 勿删
```

**首次使用：** 若弹出缺少 `const.py`，左侧选 **初始化** → **开始执行**（需联网一次）。

**固定到任务栏：** 右键 `SiglusSSU-GUI.exe` → 固定到任务栏。

## 安装与首次运行（开发者）

### 从源码运行

**前提：** Python 3.12+、[uv](https://github.com/astral-sh/uv)、Rust 工具链（构建加速扩展用）。

```bash
# 在项目根目录
uv sync
uv run siglus-ssu init   # 若提示缺少 const.py，执行一次
uv run siglus-ssu-gui    # 启动图形界面
```

### 从 PyPI 安装（发布后）

```bash
pip install siglus-ssu siglus-ssu-gui   # GUI 包名待发布时确定
siglus-ssu init
siglus-ssu-gui
```

### 首次启动

1. 启动本程序。
2. 若弹出「缺少运行时常量文件」，点击侧边栏 **初始化**，然后点击 **开始执行**。
3. 初始化成功后即可使用其他功能。

## 界面概览

```
侧边栏          主区域
────────        ─────────────────────────────
提取            输入路径  [浏览…]
编译            输出路径  [浏览…]
分析            选项（复选框、下拉框）
…               [ 开始执行 ]  [ 打开输出目录 ]
                运行日志（下方大文本框）
底部            ☐ 纯 Python 模式   常量配置 [0]
```

- **侧边栏**：切换功能；当前项高亮。
- **输入 / 输出**：通过「浏览…」选择文件或文件夹；也可直接粘贴路径。
- **开始执行**：调用对应的 `siglus-ssu` 命令。
- **打开输出目录**：任务成功后，用资源管理器打开结果所在文件夹。
- **运行日志**：显示命令输出；出错时在此查看原因。

## 通用操作

### 选择输入文件

1. 点击输入行右侧 **浏览…**。
2. 在对话框中选择目标文件（如 `Scene.pck`）。
3. 路径会显示在文本框中。

### 选择输入文件夹

部分功能（编译、批量反汇编）需要文件夹：

1. 点击 **浏览…**。
2. 选择包含 `.ss` 或 `.dat` 的工作目录。

### 指定输出位置

| 输出类型 | 操作 |
|---|---|
| 输出到文件夹 | **浏览…** → 选择或新建文件夹 |
| 输出为单个文件 | **浏览…** →「另存为」填写文件名（如 `Scene_translated.pck`） |
| 使用默认位置 | **留空输出框**；程序按 CLI 规则写到输入同目录或带时间戳的子目录 |

### 密钥来源（`--angou`）

加密场景需要正确密钥时，在「密钥来源」中可填写：

- 游戏根目录（内含 `暗号.dat` 或 `SiglusEngine.exe`）
- `angou=明文密钥`
- `key=AA BB CC …`（十六进制字节）

不确定时可先留空，若提取失败再填写游戏目录。

### 全局选项

| 选项 | 何时使用 |
|---|---|
| 纯 Python 模式 | 仅当 Rust 扩展异常、需要排查时勾选 |
| 常量配置 0/1/2 | 极少数引擎变体编译失败时，尝试改为 1 或 2 |

## 功能分步说明

### 提取（`-x`）

**把游戏场景包解开，得到可编辑内容。**

1. 侧边栏选 **提取**。
2. 操作类型选 **提取 .pck**。
3. 输入：浏览选择 `Scene.pck`。
4. 输出：浏览选择空文件夹（如 `D:\translate\work`）；留空则解到 `.pck` 旁边。
5. 需要反编译脚本时，勾选 **同时反汇编并反编译**。
6. 点击 **开始执行**。
7. 完成后点 **打开输出目录**，可见 `output_日期_时间` 文件夹及其中 `.dat`、源码等。

**还原 Gameexe.ini：** 操作类型改选 **还原 Gameexe.ini**，输入选 `Gameexe.dat`。

**仅处理已有 .dat：** 操作类型选 **反汇编 .dat**，输入选含 `.dat` 的文件夹。

---

### 编译（`-c`）

**把修改后的脚本目录打回 `.pck`。**

1. 侧边栏选 **编译**。
2. 输入：选择包含 `.ss` 和 `Gameexe.ini`（如有）的文件夹。
3. 输出：选择目标 `.pck` 路径，或选一个已存在目录（将生成 `Scene.pck`）。
4. 汉化常用：源文件编码选 **UTF-8**（若脚本为 UTF-8 保存）。
5. 点击 **开始执行**；日志末尾出现 `=== Compiling Stats ===` 表示完成。

**仅生成 Gameexe.dat：** 操作类型选 **仅编译 Gameexe.dat**，输出必须为文件夹。

---

### 分析（`-a`）

**查看文件内部结构，或对比原版与修改版是否一致。**

1. 侧边栏选 **分析**。
2. **分析单个文件**：选 `.pck` 或 `.dat`，结果在日志区显示。
3. **比较两个文件**：再选第二个文件，可选 **语义级比较**（更慢、更贴近「内容是否相同」）。
4. **台词统计**：选 `.pck`，可选输出 CSV 路径，用于评估翻译量。

---

### 图片 g00（`-g`）

| 操作 | 输入 | 输出 |
|---|---|---|
| 分析 | 单个 `.g00` | 无需 |
| 提取 | `.g00` 或文件夹 | 输出文件夹 |
| 合并 | 多个 `.g00`（添加文件） | 输出文件夹（`--o`） |
| 创建 | PNG/JPEG 或文件夹 | 输出 `.g00` 或文件夹 |

勾选 **裁剪透明边** 可去掉 PNG 多余透明区域。

---

### 音频（`-s`）

| 操作 | 说明 |
|---|---|
| 提取 | 从游戏目录批量解出 `.ogg` |
| 分析 | 查看单个 `.ovk` / `.owp` / `.nwa` 信息 |
| 编码 | 从 `.ogg` 生成引擎音频格式 |
| 播放 | 试听；可选指定 `Gameexe.dat` 以正确解密 |

---

### 视频（`-v`）

- **提取**：`.omv` → 视频文件到输出目录。
- **分析**：查看 `.omv` 头信息与结构。
- **编码**：`.ogv` → `.omv`；可参考已有 `.omv` 保持参数一致。

---

### 数据库（`-d`）

- **导出**：`.dbs` → 可读 CSV/文本到文件夹。
- **分析**：查看 `.dbs` 结构。
- **编译**：从编辑后的 CSV 重新生成 `.dbs`。

---

### 语音收集（`-k`）

**按角色整理语音文件，便于翻译或替换。**

1. 模式选 **按场景收集**。
2. 场景输入：`Scene.pck` 或场景目录。
3. 语音资源目录：游戏内 `voice`（或同类）文件夹。
4. 输出：空文件夹；执行后按角色分子目录生成 `.ogg`。
5. 只需统计数量时勾选 **仅统计不提取**。

---

### 文本映射（`-m`）

**翻译工作流核心：脚本 ↔ 表格互转。**

| 步骤 | 操作 |
|---|---|
| 1. 导出 | 选 **从 .ss 导出**，输入脚本或目录 → 生成 `.ss.csv` |
| 2. 翻译 | 用 Excel 等编辑 CSV 中的译文列 |
| 3. 写回 | 选 **写回 .ss**，输入同一 `.ss` 或目录 |

对二进制 `.dat` 字符串池：使用 **从 .dat 导出** / **写回 .dat**（对应 `--disam` / `--disam-apply`）。

---

### 引擎补丁（`-p`）

**修改 `SiglusEngine.exe`（请先备份原文件）。**

| 操作 | 用途 |
|---|---|
| 更换密钥 | 指向新 `key.txt` 或 `暗号.dat` |
| CJK 语言 | 界面字体与 locale 改为中文显示 |
| CJK + 中文路径 | 同上，并改用 `SceneZH.pck`、`GameexeZH.dat` 等路径 |
| 查看信息 | 只读，不修改文件 |
| 地域检测 | 关闭或恢复区域限制 |

输出 exe 可指定路径，或勾选 **直接覆盖原文件**（慎用）。

---

### 场景教程（`-t`）

1. 输入 `Scene.pck`。
2. 可选指定输出 JSON；留空则生成在同目录。
3. 执行后打开同目录下的 **`tutorial_viewer.html`**，在浏览器中查看剧情分支图。

---

### 执行标签（`-e`）

从指定场景标签启动游戏引擎（调试用）：

1. 引擎路径：`SiglusEngine.exe`。
2. 场景名：如 `title`（不含 `.ss`）。
3. 标签名：如 `start`（对应脚本中的 `#z start`）。

---

### 初始化（`init`）

下载或刷新 `const.py` 常量文件。一般只需运行一次；升级工具后可选 **强制覆盖** 再执行。

---

### 回编测试（`test`）

对 `.pck` 做提取→再编译→对比的自动化测试，用于验证工具链。输入选单个 `.pck` 或含多个 `.pck` 的文件夹。

---

### 语言服务器（`-lsp`）

供 VS Code / Cursor 等编辑器通过 LSP 获得 `.ss` 语法提示。点击开始后保持本程序运行；在编辑器中配置 `siglus-ssu -lsp` 作为语言服务器命令。

## 常用工作流

### 工作流 A：完整汉化（提取 → 改脚本 → 编译）

```
① 提取
   输入：原版 Scene.pck
   输出：D:\project\cn\
   勾选：反汇编并反编译

② 用编辑器修改 D:\project\cn\ 下 decompiled\*.ss 或文本映射 CSV

③（可选）文本映射 → 写回 .ss

④ 编译
   输入：D:\project\cn\
   输出：D:\project\SceneZH.pck
   编码：UTF-8

⑤（可选）引擎补丁 → CJK + 中文路径，并指向 SceneZH.pck
```

### 工作流 B：只改台词（文本映射）

```
① 文本映射 → 从 .ss 导出
② 翻译 CSV
③ 文本映射 → 写回 .ss
④ 编译 → 输出新 Scene.pck
```

### 工作流 C：提取角色语音

```
① 语音收集
   场景：Scene.pck
   语音目录：游戏 voice 文件夹
   输出：D:\voices_out\
```

### 工作流 D：对比汉化包与原版

```
① 分析 → 比较两个文件
   文件1：原版 Scene.pck
   文件2：汉化 Scene.pck
   可选：语义级比较
```

## 故障排除

| 现象 | 建议 |
|---|---|
| 双击 exe 无反应或闪退 | 确认解压了完整文件夹；勿杀毒软件隔离 `_internal`；尝试以管理员运行 |
| 提示缺少 `const.py` | 运行 **初始化**（便携版同样需要，仅首次需联网） |
| 提取 `.pck` 失败 / 乱码 | 填写 **密钥来源** 为游戏目录 |
| 编译报错编码 | 将 **源文件编码** 改为 UTF-8 或 Shift-JIS，与文件实际保存一致 |
| 编译结果与官方二进制不一致 | 汉化通常可忽略；追求逐字节一致需 CLI 高级参数 `--set-shuffle`（首版 GUI 未暴露） |
| 日志显示 `command not found` | 确认已 `uv sync` 或 `pip install`，且 `siglus-ssu` 在 PATH 中 |
| Rust 相关错误 | 勾选 **纯 Python 模式** 重试（速度较慢） |
| 反编译的 `.ss` 无法直接回编 | 属正常现象；反编译结果仅供阅读，请以原始提取的 source 或自行整理的 `.ss` 为准 |
| 引擎补丁后无法启动 | 恢复备份的 `SiglusEngine.exe`；确认补丁类型与游戏版本匹配 |

更多问题见上游 [Issues](https://github.com/Jirehlov/SiglusSceneScriptUtility/issues) 或 [manual_cn.md 故障排除章节](manual_cn.md#提示与故障排除)。
