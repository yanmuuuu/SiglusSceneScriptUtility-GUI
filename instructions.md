## 便携版（推荐）

**无需安装 Python、uv 或任何运行环境。**

### 安装步骤

1. 下载发布包中的 `SiglusSSU-GUI-portable.zip`（或解压后的文件夹）
2. 将整个 `SiglusSSU-GUI` 文件夹放到任意位置（建议桌面）
3. 双击 **`SiglusSSU-GUI.exe`** 启动

```
SiglusSSU-GUI/
├── SiglusSSU-GUI.exe    ← 双击启动
├── 启动 SiglusSSU-GUI.bat
├── 使用说明.txt
└── _internal/           ← 运行依赖，勿删
```

### 首次使用（必做）

1. 启动后若弹出 **「缺少运行时常量」**，点击 **是**
2. 左侧 **系统与工具** → **初始化**
3. 点击 **开始执行**（需联网，约数秒）
4. 日志出现成功信息后，即可使用其他功能

### 固定到任务栏

右键 `SiglusSSU-GUI.exe` → **固定到任务栏**。

## 安装与首次运行

### 从源码运行（开发者）

**前提：** Python 3.12+；推荐 [uv](https://github.com/astral-sh/uv)。

```bash
cd SiglusSceneScriptUtility-GUI
uv sync
uv run siglus-ssu init      # 若提示缺少 const.py
uv run siglus-ssu-gui       # 启动 GUI
```

### 首次启动检查清单

| 步骤 | 操作 |
|---|---|
| 1 | 启动程序，确认窗口标题为 SiglusSceneScriptUtility |
| 2 | 若提示缺少 `const.py` → 进入 **初始化** → **开始执行** |
| 3 | 状态栏显示 **就绪** 后即可使用 |

## 界面说明

### 左侧导航（三组）

| 分组 | 包含功能 | 典型用途 |
|---|---|---|
| **场景与脚本** | 提取、编译、分析、文本映射、场景教程、执行标签 | 汉化、改剧本 |
| **游戏资源** | 图片 g00、音频、视频、数据库、语音收集 | 解包/替换资源 |
| **系统与工具** | 引擎补丁、初始化、回编测试、语言服务器 | 环境配置、引擎修改 |

点击功能名切换面板；当前项会**高亮**。面板顶部有一行**灰色说明**，概括该功能用途。

### 主区域结构

每个功能页面通常包含：

1. **路径** — 输入、输出；右侧 **浏览…** 选择文件或文件夹  
2. **选项 / 操作类型** — 单选、下拉、复选框  
3. **高级选项**（部分功能）— 默认折叠，点击 **▶ 高级选项** 展开  
4. **操作按钮** — 开始执行、停止、打开输出目录  
5. **运行日志** — 下方深色区域，显示命令与输出  

窗口中间有一条**可拖动分隔条**，可拉高或压低日志区域。

### 底部栏

| 元素 | 说明 |
|---|---|
| 纯 Python 模式 | 一般勿勾选；仅 Rust 扩展异常时排查用 |
| 常量配置 0/1/2 | 极少数引擎变体编译失败时尝试 1 或 2 |
| 状态栏 | 就绪 / 运行中… / 完成 / 失败（退出码） |

### 快捷键

| 快捷键 | 作用 |
|---|---|
| `Ctrl + Enter` | 执行当前功能（等同「开始执行」） |
| `Ctrl + L` | 清空运行日志 |
| 日志区右键 | 复制、全选 |

## 通用操作

### 选择文件

1. 在 **输入** 行点击 **浏览…**
2. 在对话框中选择目标文件（如 `Scene.pck`）
3. 路径出现在文本框中；也可 **Ctrl+V 直接粘贴** 完整路径

程序会记住你上次浏览的文件夹，下次打开对话框时自动定位。

### 选择文件夹

1. 点击 **浏览…**
2. 选择或新建文件夹（如 `D:\translate\work`）

### 指定输出

| 需求 | 操作 |
|---|---|
| 输出到文件夹 | **浏览…** → 选择文件夹 |
| 输出为单个文件 | **浏览…** →「另存为」填写文件名（如 `SceneZH.pck`） |
| 使用 CLI 默认 | **留空输出框**（见各功能灰色 hint） |

### 密钥来源（`--angou`）

加密场景提取失败时，在 **密钥来源** 填写：

| 填写内容 | 示例 |
|---|---|
| 游戏根目录 | `D:\Games\MyGame\`（含 `暗号.dat` 或引擎） |
| 明文字符串 | `angou=你的密钥` |
| 十六进制 | `key=AA BB CC DD …` |

不确定时先留空；失败后再填游戏目录。

### 执行与查看结果

1. 填好必填项后点击 **开始执行**（或 `Ctrl+Enter`）
2. 运行中：按钮变灰、右侧出现**滚动进度条**、日志实时追加
3. 完成后：状态栏 **完成**；点 **打开输出目录** 在资源管理器中查看
4. 失败：看日志末尾错误信息；状态栏显示退出码

### 停止任务

长时间任务可点 **停止**；关闭窗口时若任务仍在运行会提示确认。

## 功能分步说明

以下按侧边栏分组排列。每节包含：**用途 → 逐步操作 → 注意事项**。

---

### 提取（`-x`）

**用途：** 从 `.pck` 解包场景；将 `.dat` 反汇编为脚本；还原 `Gameexe.ini`。汉化第一步。

#### A. 提取 .pck（最常用）

| 步骤 | 操作 |
|---|---|
| 1 | 左侧 **场景与脚本** → **提取** |
| 2 | 操作类型选 **提取 .pck** |
| 3 | **输入** → 浏览选择 `Scene.pck`（或其他 `.pck`） |
| 4 | **输出目录** → 选空文件夹，如 `D:\work\cn\`；或留空（解到 `.pck` 同目录） |
| 5 | 需要可编辑脚本时，勾选 **同时反汇编并反编译** |
| 6 | **开始执行** |
| 7 | 完成后 **打开输出目录**，找到 `output_日期_时间` 文件夹 |

**输出内容：** `.dat`、反编译的 `.ss`、资源文件等（视选项而定）。

#### B. 反汇编已有 .dat

| 步骤 | 操作 |
|---|---|
| 1 | 操作类型选 **反汇编 .dat（目录或 .pck）** |
| 2 | **输入** 选含 `.dat` 的文件夹，或原始 `.pck` |
| 3 | 执行 |

#### C. 还原 Gameexe.ini

| 步骤 | 操作 |
|---|---|
| 1 | 操作类型选 **还原 Gameexe.ini** |
| 2 | **输入** 选 `Gameexe.dat` 或其所在目录 |
| 3 | 执行 |

**注意：** 提取失败或乱码时，在 **密钥来源** 填游戏根目录后重试。

---

### 编译（`-c`）

**用途：** 将修改后的脚本目录打包回 `.pck`。汉化最后一步。

| 步骤 | 操作 |
|---|---|
| 1 | **编译** |
| 2 | 操作类型：**标准编译**（一般选这个） |
| 3 | **输入目录** → 含 `.ss` 和 `Gameexe.ini` 的工作目录 |
| 4 | **输出** → 目标 `.pck` 完整路径，或选一个文件夹（生成 `Scene.pck`） |
| 5 | **源文件编码** → 汉化常用 **UTF-8**（与你在编辑器中的保存编码一致） |
| 6 | **开始执行** |
| 7 | 日志末尾出现 `=== Compiling Stats ===` 表示完成 |

**仅编译 Gameexe.dat：** 操作类型改 **仅编译 Gameexe.dat**，**输出** 必须为文件夹。

**高级选项（一般不用）：** 展开 **▶ 高级选项** 可设置增量缓存 `--tmp` 等；`--debug` 与 `--tmp` 不能同时勾选。

---

### 分析（`-a`）

**用途：** 查看包内结构、对比两个文件、统计台词量。结果在**日志区**，不单独弹窗。

#### 分析单个文件

| 步骤 | 操作 |
|---|---|
| 1 | **分析** → 操作类型 **分析单个文件** |
| 2 | **输入文件 1** → 选 `.pck`、`.dat` 等 |
| 3 | 可选勾选 **写出反汇编** |
| 4 | 执行 → 在日志中阅读结构信息 |

#### 比较两个文件

| 步骤 | 操作 |
|---|---|
| 1 | 操作类型 **比较两个文件** |
| 2 | 分别选择原版与修改版（如两个 `Scene.pck`） |
| 3 | 需要内容级对比时勾选 **语义级比较**（更慢） |
| 4 | 执行 → 日志显示差异摘要 |

#### 台词统计导出 CSV

| 步骤 | 操作 |
|---|---|
| 1 | 操作类型 **台词统计导出 CSV** |
| 2 | 选 `.pck`；**输出 CSV** 可留空（默认路径）或指定 |
| 3 | 执行 → 用 Excel 打开 CSV 评估翻译量 |

---

### 文本映射（`-m`）

**用途：** 脚本与 CSV 表格互转，翻译工作流核心。

| 操作 | 步骤概要 |
|---|---|
| **从 .ss 导出** | 选脚本或目录 → 生成 `.ss.csv` |
| **写回 .ss** | 翻译 CSV 后，同操作选 **写回 .ss**，输入原 `.ss` 或目录 |
| **从 .dat 导出** | 对二进制 `.dat` 字符串池导出 |
| **写回 .dat** | 翻译后写回 `.dat` |

**典型顺序：** 导出 → Excel 编辑译文列 → 写回 → **编译** 打 `.pck`。

---

### 场景教程（`-t`）

| 步骤 | 操作 |
|---|---|
| 1 | **场景教程** |
| 2 | **输入** → `Scene.pck` |
| 3 | **输出 JSON** 可留空（默认同目录 `<名>.tutorial.json`） |
| 4 | 执行后 **打开输出目录** |
| 5 | 用浏览器打开同目录 **`tutorial_viewer.html`**，加载 JSON 查看剧情分支图 |

### 执行标签（`-e`）

| 字段 | 填写示例 |
|---|---|
| 引擎路径 | `D:\Game\SiglusEngine.exe` |

| 操作 | 输入 | 输出 | 备注 |
|---|---|---|---|

| **提取** | `.g00` 或文件夹 | 输出文件夹 | 可勾选裁剪透明边 |

### 语音收集（`-k`）

**用途：** 按角色整理语音文件。

| 步骤 | 操作 |
|---|---|

## 常用工作流

### 工作流 A：完整汉化

```
① 提取
   输入：Scene.pck
   输出：D:\project\cn\
   勾选：同时反汇编并反编译

② 用编辑器修改 D:\project\cn\ 下脚本

③（可选）文本映射 → 导出 CSV → 翻译 → 写回

④ 编译
   输入：D:\project\cn\
   输出：D:\project\SceneZH.pck
   编码：UTF-8

⑤（可选）引擎补丁 → CJK + 中文路径
```

### 工作流 B：只改台词

```
文本映射：从 .ss 导出 → 翻译 CSV → 写回 .ss → 编译
```

### 工作流 C：提取角色语音

```
语音收集：Scene.pck + voice 目录 → 输出空文件夹
```

### 工作流 D：对比汉化包与原版

```
分析 → 比较两个文件 → 可选语义级比较
```

## 故障排除

| 现象 | 建议 |
|---|---|
| 界面卡顿、日志刷屏慢 | v0.2 已优化批量刷新；仍慢可 **清空** 日志后重试 |
| 双击 exe 无反应 | 确认完整解压；杀毒勿隔离 `_internal` |
| 缺少 `const.py` | **初始化**（需联网） |
| 提取失败 / 乱码 | **密钥来源** 填游戏根目录 |
| 编译编码错误 | **源文件编码** 改为 UTF-8 或 Shift-JIS |
| 命令找不到 | 开发者：`uv sync`；便携版：勿移动 exe 出文件夹 |
| Rust 错误 | 勾选 **纯 Python 模式**（较慢） |
| 反编译 .ss 无法直接回编 | 正常；请用原始 source 或自行整理的 `.ss` |
| 补丁后引擎无法启动 | 恢复备份 exe |

更多见 [manual_cn.md](manual_cn.md) 与上游 [Issues](https://github.com/Jirehlov/SiglusSceneScriptUtility/issues)。
