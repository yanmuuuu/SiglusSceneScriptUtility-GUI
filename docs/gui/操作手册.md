# SiglusSceneScriptUtility GUI · 操作手册

本手册内嵌于图形界面，面向 **SiglusEngine 游戏完整汉化**。示例路径均使用抽象写法（如「你的游戏目录」），请替换为你本机实际位置。

---

## 阅读顺序（汉化推荐）

1. **完整汉化工作流** — 从提取到打补丁的全程（必读）
2. **初始化** — 首次使用先完成
3. **提取** → **文本映射** → **编译** → **引擎补丁** — 汉化核心四步
4. **全局选项** / **故障排除** — 遇到问题时查阅

## 左侧功能一览

### 场景与脚本（汉化主战场）

| 功能 | 用途 |
|---|---|
| 提取 | 从 Scene.pck 解包脚本与资源 |
| 编译 | 把改好的脚本目录打回 .pck |
| 分析 | 查看结构、对比原版与汉化版 |
| 文本映射 | CSV 表格方式翻译台词 |
| 场景教程 | 生成剧情分支图（辅助校对） |
| 执行标签 | 从指定场景启动引擎（调试） |

### 游戏资源

| 功能 | 用途 |
|---|---|
| **资源浏览** | 选根目录、分类筛选、缩略图/列表、图片与音频预览 |
| 图片 g00 | 立绘、背景等资源提取与回编 |
| 音频 / 视频 / 数据库 | 其他资源解包与回编 |
| 语音收集 | 按角色整理语音文件 |

## 每个功能面板怎么用

1. 左侧点击功能名
2. 按灰色说明与字段提示填写路径
3. 不确定时点右上角 **「教程」** 按钮查看该功能全文
4. 点击 **开始执行**（或 Ctrl+Enter）
5. 在下方 **运行日志** 查看进度与错误
6. 成功后点 **打开输出目录**
