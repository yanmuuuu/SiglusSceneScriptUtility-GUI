# 资源浏览 · 使用教程

像 **FModel** 一样：先选一个**根目录**，再按类型筛选、预览资源。专为 Siglus 游戏目录与汉化工作目录设计。

---

## 这个功能做什么

- 扫描根目录（可选包含子文件夹）— **懒加载**：先只列出文件名与类型，大小与预览按需读取
- **分类筛选**：全部 / 图片 / 音频 / 视频 / 脚本 / 其他
- **文件名搜索**
- **列表**或**缩略图**浏览图片（`.g00` `.png` 等）
- 右侧**预览**：大图、文本摘要、场景包说明
- **试听** `.ogg` `.owp` `.nwa`（需系统安装 ffplay/ffmpeg）
- **打开** `.omv` 视频（提取后用系统播放器）
- **跳转**到提取 / 图片 g00 / 音频等面板并自动填入路径

---

## 逐步操作

1. 左侧 **游戏资源** → **资源浏览**
2. **根目录** → **选择目录…** → 你的游戏目录 或 汉化工作目录
3. 勾选 **包含子文件夹**（默认开）
4. **分类** 选「图片」找立绘，或「脚本」找 `.pck` `.ss`
5. 在左侧列表**单击**文件 → 右侧仅显示文件信息（路径、类型）；**双击**或点 **预览** 才加载内容
6. 图片分类下可切换 **缩略图** 网格（5 列，**仅加载可见区域**缩略图）
7. 音频文件点 **播放/打开** 试听
8. `.pck` 点 **跳转功能** → 自动打开「提取」并填好路径

---

## 字段说明

| 字段 | 说明 |
|---|---|
| 根目录 | 任意文件夹；会记住上次路径 |
| 包含子文件夹 | 关闭则只扫描当前层 |
| 分类 | 按扩展名归类 Siglus 资源 |
| 搜索 | 过滤相对路径或文件名 |
| 列表 / 缩略图 | 缩略图仅在「全部/图片」下推荐 |

---

## 支持预览的类型

| 类型 | 扩展名 | 预览方式 |
|---|---|---|
| 图片 | .g00 .g01 .png .jpg … | 缩略图 + 大图（G00 自动解出 PNG） |
| 音频 | .ogg .wav .owp .nwa | ffplay 试听 |
| 视频 | .omv .ogv .mp4 | 提取/打开系统播放器 |
| 文本 | .ss .ini .csv .json … | 前若干行文本 |
| 场景包 | .pck | 文件信息 + 跳转提取 |
| 语音包 | .ovk | 说明 + 建议用音频/语音收集 |

---

## 快捷操作

| 操作 | 说明 |
|---|---|
| 预览 (Enter) | 刷新右侧预览 |
| 播放/打开 | 音频试听或视频打开 |
| 停止播放 | 停止 ffplay |
| 打开所在目录 | 资源管理器定位文件 |
| 跳转功能 | 跳到对应 CLI 面板 |
| F5 | 重新扫描 |

## 汉化工作流中的位置

| 场景 | 用法 |
|---|---|
| 找 Scene.pck | 分类「脚本」→ 跳转提取 |
| 核对立绘是否齐全 | 分类「图片」+ 缩略图 |
| 听 BGM/语音样本 | 分类「音频」+ 播放 |
| 浏览解包结果 | 根目录选 output_xxx 文件夹 |

---

## 常见错误

| 情况 | 处理 |
|---|---|
| 图片预览失败 | 便携版需含 Pillow；G00 损坏会提示 |
| 无法播放音频 | 安装 [ffmpeg](https://ffmpeg.org/) 并确保 ffplay 在 PATH |
| 扫描很慢 | 已改为懒加载：先只扫文件名；缩小根目录或关闭子文件夹 |
| 缩略图一直转圈 | 仅可见区域加载；G00 解码较慢，滚动后稍等 |
| 文件过多 | 默认最多 8000 条，请用搜索缩小范围 |
