# 数据库 · 使用教程

导出、分析、编译游戏 `.dbs` 数据库（部分游戏 UI 文本、配置表在此）。

---

## 支持的主要文件

| 扩展名 | 说明 |
|---|---|
| `.dbs` | Siglus 数据库二进制 |
| `.csv` | 导出结果；编译模式输入目录中的表格 |

---

## 操作类型

| 操作 | 作用 |
|---|---|
| 导出 --x | .dbs → CSV 到文件夹 |
| 分析 --a | 结构信息（日志） |
| 编译 --c | CSV 目录 → 新 .dbs |

---

## 逐步操作：导出数据库

1. **游戏资源** → **数据库**
2. **操作** → **导出 --x**
3. **输入** → .dbs 文件或含多个 .dbs 的文件夹
4. **输出** → 输出文件夹（**必填**）
5. **开始执行** → 得到 CSV 可编辑

### 编译回 .dbs

1. **操作** → **编译 --c**
2. **输入** → 编辑后的 CSV 目录
3. **输出** → 另存为 .dbs 或选文件夹
4. **数据库类型** 按需填写（高级）

---

## 字段说明

| 字段 | 导出 | 分析 | 编译 |
|---|---|---|---|
| 输入 | .dbs/目录 | .dbs | CSV/目录 |
| 输出 | 文件夹必填 | 无 | 必填 |
| 数据库类型 | 隐藏 | 隐藏 | 可选 --type |

---

## 汉化相关

- 若游戏菜单/系统文本在 .dbs 而非 Scene.pck，需**额外汉化此库**
- 流程：导出 CSV → 翻译 → 编译 → 替换游戏内 .dbs
- 与 Scene 脚本汉化**并行**，最后一起替换进游戏

---

## 常见错误

| 情况 | 处理 |
|---|---|
| 编译类型错误 | 填写正确数据库类型 |
| CSV 改乱 | 从重新导出开始 |
