# 分析 · 使用教程

查看包结构、对比原版与汉化版、统计台词量。结果在**运行日志**中显示，不单独弹窗。

---

## 这个功能做什么

- 分析单个 `.pck` / `.dat` 等结构
- 比较两个文件差异（汉化质检）
- 导出台词统计 CSV（评估翻译量）
- 分析 `Gameexe.dat`

---

## 支持的主要文件

| 扩展名 | 用途 |
|---|---|
| `.pck` | 场景包分析、对比、台词统计 |
| `.dat` | 单文件分析 |
| `Gameexe.dat` | 专用分析模式 |
| 输出 `.csv` | 台词统计导出 |

---

## 逐步操作

### 分析单个文件

1. **操作类型** → 分析单个文件
2. **输入文件 1** → 选文件 → 你的游戏目录\Scene.pck
3. 可选勾选 **写出反汇编**
4. **开始执行** → 在日志阅读结构信息

### 比较两个文件（汉化质检）

1. **操作类型** → 比较两个文件
2. **输入文件 1** → 原版 Scene.pck
3. **输入文件 2** → 汉化 SceneZH.pck
4. 需要内容级差异时勾选 **语义级比较（较慢）**
5. 执行 → 日志显示差异摘要

### 台词统计导出 CSV

1. **操作类型** → 台词统计导出 CSV
2. **输入文件 1** → .pck
3. **输出 CSV** 可留空（默认路径）或另存为指定 .csv
4. 用表格软件打开 CSV 评估工作量

### 分析 Gameexe.dat

1. **操作类型** → 分析 Gameexe.dat
2. **输入文件 1** → Gameexe.dat
3. 可选 **密钥来源**

---

## 字段说明

| 字段 | 何时显示 | 支持文件 | 必填 | 结果 |
|---|---|---|---|---|
| 输入文件 1 | 始终 | .pck .dat 等 | 是 | 日志分析 |
| 输入文件 2 | 仅「比较」 | 同左 | 是 | 对比报告 |
| 输出 CSV | 仅「台词统计」 | .csv | 否 | CSV 文件 |
| 写出反汇编 | 单文件/比较 | — | 否 | 额外 disasm 输出 |
| 语义级比较 | 仅「比较」 | — | 否 | 更细的文本差异 |
| 密钥来源 | 加密包 | 游戏目录等 | 否 | 正确解密 |

---

## 汉化中的用法

- 编译完成后 **对比原版与汉化 pck**，确认改动符合预期
- 提取前 **台词统计** 评估工期
- 不解包快速看 **pck 内场景列表**（日志）

---

## 常见错误

| 情况 | 处理 |
|---|---|
| 日志太长 | Ctrl+L 清空后重新跑单项 |
| 解密失败 | 填密钥来源 |
| 比较很慢 | 语义级比较对大包耗时久，耐心等待 |
