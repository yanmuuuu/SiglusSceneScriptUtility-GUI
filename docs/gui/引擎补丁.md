# 引擎补丁 · 使用教程

修改 `SiglusEngine.exe`：中文界面、中文路径、换密钥等。**汉化发布前关键一步**。

---

## 警告

- **务必先备份** 原版 SiglusEngine.exe
- 补丁错误可能导致游戏无法启动
- 「直接覆盖原文件」仅在确认备份后使用

---

## 支持的主要文件

| 类型 | 说明 |
|---|---|
| SiglusEngine.exe | 游戏主程序 |
| 密钥来源 | key.txt、暗号.dat、Scene.pck 等（换密钥模式） |
| 输出 exe | 补丁后的新文件 |

---

## 汉化常用操作

### CJK 中文界面

1. **系统与工具** → **引擎补丁**
2. **操作** → **CJK 语言 --lang cjk**
3. **SiglusEngine.exe** → 你的游戏目录\SiglusEngine.exe
4. **输出 exe** → 另存为 你的游戏目录\SiglusEngine_zh.exe
5. **开始执行**
6. 用新 exe 启动游戏

### CJK + 中文路径（加载 SceneZH.pck）

1. **操作** → **CJK + 中文路径 --lang cjk-path**
2. 其余同上
3. 游戏配置或启动参数需指向中文名 pck（因游戏而异）

---

## 字段说明

| 字段 | 何时需要 | 说明 |
|---|---|---|
| 操作 | 始终 | 见下表 |
| SiglusEngine.exe | 始终 | **必填** 待补丁 exe |
| 密钥来源 | 更换密钥 | key.txt / 暗号.dat / Scene.pck |
| 输出 exe | 建议填 | 另存为新 exe |
| 直接覆盖 | 可选 | --inplace，覆盖原 exe |
| 地域检测 | --loc 模式 | 关闭(0) 或 恢复(1) |

### 操作类型

| 操作 | 用途 |
|---|---|
| 更换密钥 --altkey | 换游戏解密密钥 |
| CJK 语言 | 界面中文显示 |
| CJK + 中文路径 | 中文 UI + 中文路径 pck |
| 查看信息 --info | 只读，不写文件 |
| 地域检测 --loc | 区域限制开关 |

---

## 你会得到什么

- 新 exe（或覆盖原 exe）
- 「查看信息」仅在日志显示可补丁槽位

---

## 汉化流程中的位置

```
编译 SceneZH.pck → 引擎补丁 → 用新 exe 启动测试
```

若仍加载原版 Scene.pck，需替换文件或改配置。

---

## 常见错误

| 情况 | 处理 |
|---|---|
| 补丁后闪退 | 恢复备份；换 cjk 而非 cjk-path 试 |
| 仍日文 UI | 确认用补丁后 exe 启动 |
| 中文路径无效 | 确认选 cjk-path；pck 文件名与游戏一致 |
