# 视频 · 使用教程

> **便携版用户：** 无需安装 Python/ffmpeg；下载与启动见 [instructions.md](../../instructions.md#便携版推荐)。

处理游戏过场等 `.omv` 视频：提取、分析、从 `.ogv` 编码回包。

---

## 支持的主要文件

| 扩展名 | 说明 |
|---|---|
| `.omv` | Siglus 视频（提取/分析/编码输出） |
| `.ogv` | 编码模式输入 |

---

## 操作类型

| 操作 | 输入 | 输出 |
|---|---|---|
| 提取 --x | .omv | 输出文件夹 |
| 分析 --a | .omv | 日志 |
| 编码 --c | .ogv | .omv 或文件夹 |

---

## 逐步操作：提取视频

1. **游戏资源** → **视频**
2. **操作** → **提取 --x**
3. **输入** → 选文件 → 你的游戏目录\movie\xxx.omv
4. **输出** → 选文件夹（必填）
5. **开始执行**

---

## 字段说明

| 字段 | 操作 | 必填 | 说明 |
|---|---|---|---|
| 输入 | 全部 | 是 | .omv 或 .ogv（编码） |
| 输出 | 提取/编码 | 是 | 文件夹或另存为 .omv |
| 参考 omv | 仅编码 | 否 | 保持与参考相同编码参数 |

---

## 汉化相关

- 带字幕的 OP/ED 视频汉化需单独字幕轨或替换视频，本工具只做**解包/回封**
- 纯文字汉化一般**不涉及**本功能

---

## 常见错误

| 情况 | 处理 |
|---|---|
| 编码失败 | 检查 .ogv 格式；填参考 omv |
| 提取文件很大 | 确保输出磁盘空间充足 |
