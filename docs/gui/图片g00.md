# 图片 g00 · 使用教程

> **便携版用户：** 无需安装 Python/ffmpeg；下载与启动见 [instructions.md](../../instructions.md#便携版推荐)。

处理 Siglus **立绘、背景** 等 `.g00` 图片：分析、提取 PNG、合并、从图片创建。

---

## 支持的主要文件

| 扩展名 | 用途 |
|---|---|
| `.g00` | Siglus 图片包（输入/输出） |
| `.png` `.jpeg` `.jpg` | 提取结果或创建输入 |
| JSON 布局 | 创建模式 type2 等 |

---

## 操作类型概览

| 操作 | 输入 | 输出 | 汉化相关 |
|---|---|---|---|
| 分析 --a | 单个 .g00 | 无（看日志） | 查格式 |
| 提取 --x | .g00 或文件夹 | 输出文件夹 | 抠立绘/背景 |
| 合并 --m | 文件列表 | 输出文件夹 | 拼图层 |
| 创建 --c | 图片或文件夹 | .g00 或文件夹 | 替换资源 |

切换操作后，界面会**自动隐藏**无关字段。

---

## 逐步操作：提取立绘/背景

1. 左侧 **游戏资源** → **图片 g00**
2. **操作** → **提取 --x**
3. **输入** → 选文件或文件夹（你的游戏目录\cg\xxx.g00 或含多个 g00 的目录）
4. **输出目录** → **必填**，选空文件夹
5. 可选 **裁剪透明边（--trim）**
6. **开始执行** → 输出目录得 PNG/JPEG

---

## 字段说明（按操作）

### 分析 --a

| 字段 | 说明 |
|---|---|
| 输入 | 单个 .g00 |
| 输出 | 无 |
| 结果 | 日志中类型、尺寸、层信息 |

### 提取 --x

| 字段 | 必填 | 说明 |
|---|---|---|
| 输入 | 是 | .g00 或含 g00 的文件夹 |
| 输出目录 | **是** | PNG/JPEG 输出位置 |
| 裁剪透明边 | 否 | 去掉透明/纯色边 |

### 合并 --m

| 字段 | 必填 | 说明 |
|---|---|---|
| 合并文件列表 | **是** | 点「添加文件…」多个 .g00 |
| 输出目录 | **是** | 合并 PNG 输出 |
| 裁剪透明边 | 否 | 合并后裁剪 |

### 创建 --c

| 字段 | 必填 | 说明 |
|---|---|---|
| 输入 | 是 | PNG/JPEG/JSON 或文件夹 |
| 输出 | 建议填 | 另存为 .g00 或选文件夹 |
| g00 类型 | 否 | --type N，高级 |
| 参考 g00 | 否 | 保持与参考相同参数 |

---

## 汉化注意

- 改图后需**回编 g00** 并替换游戏内原文件；路径因游戏而异
- 复杂立绘（多层）合并/创建较专业，可参考 manual_cn.md
- 仅改文字通常**不需要**动 g00

---

## 常见错误

| 情况 | 处理 |
|---|---|
| 提取全黑/透明 | 检查 g00 类型；试不勾选 trim |
| 创建失败 | 需 Pillow；便携版一般已含 |
| 合并缺文件 | 列表至少添加一个 g00 |
