# 场景教程 · 使用教程

从 Scene.pck 生成**静态剧情分支图** JSON，配合浏览器查看器校对路线（汉化辅助，非必须）。

---

## 这个功能做什么

- 分析场景跳转关系，输出 `.tutorial.json`
- 同目录可使用 `tutorial_viewer.html` 可视化剧情图

---

## 支持的主要文件

| 类型 | 说明 |
|---|---|
| 输入 | `Scene.pck`（或你的游戏目录下场景包） |
| 输出 | `.json`（可留空，默认同目录 `<包名>.tutorial.json`） |

---

## 逐步操作

1. 左侧 **场景与脚本** → **场景教程**
2. **输入 Scene.pck** → 选文件 → 你的游戏目录\Scene.pck
3. **输出 JSON** 可留空，或另存为 `汉化工作目录\scene_graph.json`
4. **开始执行**
5. 完成后 **打开输出目录**
6. 用浏览器打开同目录 **tutorial_viewer.html**，加载生成的 JSON

---

## 字段说明

| 字段 | 支持文件 | 必填 | 留空 | 结果 |
|---|---|---|---|---|
| 输入 Scene.pck | .pck | **是** | — | — |
| 输出 JSON | .json | 否 | 默认在 pck 旁生成 | .tutorial.json |

---

## 汉化中的用途

- 检查分支、选项是否都覆盖翻译
- 查找漏翻场景
- 不向游戏写入任何文件，**只读分析**

---

## 常见错误

| 情况 | 处理 |
|---|---|
| 浏览器打不开图 | 确认 viewer 与 JSON 路径；用本地文件打开 |
| 图不完整 | 静态分析保守，极复杂跳转可能不全 |
