# 初始化 · 使用教程

> **便携版用户：** 双击 [instructions.md](../../instructions.md#便携版推荐) 中的「下载 SiglusSSU-GUI.bat」即可安装到桌面，无需 Python。

下载或刷新运行时常量 **const.py**。**首次使用任何功能前必须完成**（汉化起点）。

---

## 这个功能做什么

- 从上游仓库下载校验过的 `const.py`
- 内含操作码表、密钥推导参数等，提取/编译依赖此文件

---

## 何时需要执行

| 情况 | 是否执行 |
|---|---|
| 首次打开程序，提示缺少常量 | **必须** |
| 升级 siglus-ssu / 便携版后 | 建议执行一次 |
| 日常使用已正常 | 不需要 |

---

## 逐步操作

1. 左侧 **系统与工具** → **初始化**
2. 首次使用：**直接开始执行**（保持默认）
3. 升级后可选勾选 **强制覆盖已有常量（--force）**
4. **Git 引用** 一般**留空**（高级用户可填版本标签如 v0.3.7）
5. **开始执行**（需**联网**，数秒）
6. 日志显示成功 → 状态栏就绪 → 可进行 **提取**

---

## 字段说明

| 字段 | 必填 | 说明 |
|---|---|---|
| 强制覆盖 --force | 否 | 已有 const 时仍重新下载 |
| Git 引用 --ref | 否 | 指定下载版本，与 force 配合 |

无输入/输出路径。

---

## 你会得到什么

- 用户数据目录中的 const.py（Windows 一般在 %APPDATA%\siglus-ssu\）
- 仓库源码运行也会使用捆绑的 const；init 刷新用户副本

---

## 汉化流程中的位置

```
第 0 步：初始化 → 第 1 步：提取 Scene.pck → …
```

未初始化时提取/编译会报错或弹窗引导。

---

## 常见错误

| 情况 | 处理 |
|---|---|
| 下载失败 | 检查网络；稍后重试 |
| 仍提示缺少 | 勾选 force 再执行 |
| 离线环境 | 需在有网机器 init 后复制 const，见 manual |
