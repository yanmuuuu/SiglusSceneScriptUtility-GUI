# 故障排除

汉化过程中常见问题与处理。仍无法解决时，可打开 **instructions.md** 或上游手册 **manual_cn.md**。

---

## 启动与环境

| 现象 | 原因 | 处理 |
|---|---|---|
| bat 提示需要 Python 3.12+ | 未打包便携版且系统 Python 过旧 | 用 dist\SiglusSSU-GUI\SiglusSSU-GUI.exe，或双击 **环境配置.bat**（自动装 uv + Python 3.12 + 依赖） |
| 首次克隆仓库无法启动 | 尚未创建 `.venv` | 双击 **环境配置.bat**，或直接 **启动 SiglusSSU-GUI.bat**（会自动配置） |
| 双击 exe 无反应 | 未完整解压，缺 _internal | 保持整个文件夹完整 |
| 弹出多个 GUI 窗口 | 旧版便携包多进程问题 | 使用本仓库最新打包的 exe |
| 缺少 const.py | 未初始化 | **初始化** → 开始执行（联网） |

## 提取与密钥

| 现象 | 原因 | 处理 |
|---|---|---|
| 提取失败 / 乱码 / 解密错误 | 场景包加密，密钥未解析 | **密钥来源** 填你的游戏目录（含 暗号.dat 或引擎） |
| 找不到 Scene.pck | 包名不同或内嵌 exe | 在游戏目录搜索 .pck；部分游戏脚本在 exe 内，需专门处理 |
| 输出不知道在哪 | 输出目录留空 | 默认在输入 .pck 同目录；用 **打开输出目录** |

### 密钥来源怎么填

| 填写内容 | 示例 |
|---|---|
| 游戏根目录 | 你的游戏目录 |
| 明文字符串 | angou=密钥文本 |
| 十六进制 key | key=AA BB CC …（见 manual） |

## 编译与编码

| 现象 | 原因 | 处理 |
|---|---|---|
| 编译报编码错误 | .ss 保存编码与「源文件编码」不一致 | 统一用 UTF-8 保存，编译选 UTF-8 |
| 编译报语法/宏错误 | 改坏了 .ss 结构 | 对比原版 diff；用文本映射而非手改结构行 |
| 编译报 `keylist.clear` 等未知宏 | Steam/HD 版部分系统脚本（如 CLANNAD `_system_language.ss`） | **回编测试** 已通过即工具链正常；汉化请只编译你改过的场景，勿整包重编原版系统脚本 |
| Compiling Stats 不出现就失败 | 路径或权限问题 | 检查输入目录是否含 .ss；输出路径可写 |
| --debug 与 --tmp 冲突 | 两项不能同时开 | 只保留其一（见编译教程） |

## 汉化包进游戏

| 现象 | 原因 | 处理 |
|---|---|---|
| 仍是日文 | 未加载汉化 pck | 替换 Scene.pck 或打 cjk-path 补丁加载 SceneZH.pck |
| 中文方块/乱码 | 未打 CJK 补丁 | **引擎补丁** → CJK 或 CJK+中文路径 |
| 补丁后无法启动 | 补丁不匹配或损坏 | 恢复备份 exe |
| 部分 UI 仍日文 | 文本在别的文件（db、g00 等） | 需额外处理对应资源 |

## 性能与界面

| 现象 | 处理 |
|---|---|
| 日志刷屏卡顿 | v0.2+ 已批量刷新；可 **清空** 日志后继续 |
| 任务很慢 | 大包正常；可勾选 legacy 会变慢，一般不建议 |
| 停止无效 | 点 **停止** 或关闭窗口确认终止 |

## 文本映射 CSV

| 现象 | 处理 |
|---|---|
| 写回后没变化 | 确认选对「写回 .ss」且 CSV 与 .ss 对应 |
| Excel 乱码 | CSV 用 UTF-8 保存 |
| 改了不该改的列 | 只改译文列，恢复 CSV 从重新导出开始 |

---

## 资源类（g00 / 音频等）

| 现象 | 处理 |
|---|---|
| g00 提取空白 | 换操作类型；检查是否 type 特殊 |
| 音频解密失败 | 指定 Gameexe.dat |
| 资源浏览无法播放 | 便携版需完整 `ffmpeg` 文件夹；重新下载 zip；或首次播放时等待自动下载 |
| 音频组件下载很慢 | 网络访问 GitHub/gyan.dev 较慢 | 可手动 `winget install Gyan.FFmpeg.Essentials` 后重新打包，或保持文件夹完整等待后台下载完成 |
| 反编译 .ss 不能直接当源码编译 | 正常；用 decompiled 里工具生成的或自己整理的 .ss |
