# 音频 · 使用教程

提取、分析、编码、播放 Siglus 音频（`.ovk` `.owp` `.nwa` `.ogg`）。

**便携版**已内置 ffmpeg/ffplay；在「资源浏览」试听或在「音频 → 播放」时无需单独安装。

---

## 支持的主要文件

| 扩展名 | 说明 |
|---|---|
| `.ovk` | 语音包 |
| `.owp` | 音效/BGM 包 |
| `.nwa` | BGM |
| `.ogg` | 编码模式输入 |
| `Gameexe.dat` | 提取/播放时辅助解密 |

---

## 操作类型

| 操作 | 用途 |
|---|---|
| 提取 --x | 解包到文件夹 |
| 分析 --a | 查看头信息（日志） |
| 编码 --c | .ogg → 引擎格式 |
| 播放 --play | 终端试听 BGM（需 Gameexe 循环点表） |

---

## 逐步操作：提取语音/BGM

1. **游戏资源** → **音频**
2. **操作** → **提取 --x**
3. **输入** → 你的游戏目录\voice 或单个 .ovk
4. **输出目录** → **必填**
5. **Gameexe.dat** 可选 → 你的游戏目录\Gameexe.dat（解密失败时填写）
6. **密钥来源** 按需填写
7. **开始执行**

从 **资源浏览** 选中音频文件后点 **跳转功能**，会自动打开本面板并填入路径。

---

## 字段说明

| 字段 | 何时显示 | 必填 | 结果 |
|---|---|---|---|
| 输入 | 始终 | 是 | 源音频 |
| 输出目录 | 提取/编码 | 提取编码必填 | 解出 .ogg/.wav 等 |
| Gameexe.dat | 提取/播放 | 否 | 正确解密 |
| 密钥来源 | 提取等 | 否 | 解密失败时用 |

### 分析 --a

- 输入：单个 .ovk / .owp / .nwa
- 无输出，看日志

### 编码 --c

- 输入：.ogg 文件
- 输出目录：必填

### 播放 --play

- 输入：文件或文件夹
- Gameexe.dat：可选，用于 BGM 循环点

---

## 汉化相关

- **替换配音**需提取→替换→编码→放回游戏，工作量大
- **仅字幕汉化**通常不动音频
- **语音收集**（另一功能）按角色整理已有语音，便于听译校对

---

## 常见错误

| 情况 | 处理 |
|---|---|
| 提取静音/失败 | 指定 Gameexe.dat 与密钥 |
| 播放无 ffplay | **便携版**：确认 `ffmpeg` 文件夹完整；**源码**：运行 `scripts/fetch_ffmpeg.py` 或安装 [ffmpeg](https://ffmpeg.org/) |
| --trim 失败 | 裁剪 .owp 需要 ffmpeg；.nwa 不依赖 ffmpeg |
